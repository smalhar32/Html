This is basics of Html.

HTML stands for Hyper Text Markup Language. It is used to design web pages using a markup language. HTML is the standard markup language for creating Web pages.

Most Used HTML Tags

  • <html> - Root element of an HTML document
  • <head> - Container for metadata
  • <title> - Title of the document
  • <body> - Body of the document
  • <h1> to <h6> - Heading tags
  • <p> - Paragraph
  • <a> - Anchor (link)
  • <img> - Image
  • <ul> - Unordered list
  • <ol> - Ordered list
  • <li> - List item
  • <table> - Table
  • <tr> - Table row
  • <td> - Table cell
  • <th> - Table header cell
  • <form> - Form
  • <input> - Input field
  • <button> - Button
  • <div> - Division or section
  • <span> - Inline container

Advanced HTML Tags and Elements

  • <canvas> - Used to draw graphics on the fly via scripting (usually JavaScript)
  • <svg> - Used to define vector-based graphics for the web
  • <video> - Used to embed video content
  • <audio> - Used to embed sound content
  • <source> - Specifies multiple media resources for media elements (<video> and <audio>)
  • <picture> - Contains zero or more <source> elements and one <img> element to provide multiple image resources for responsive design
  • <track> - Used to specify text tracks for <video> and <audio> elements
  • <details> - Used to create a disclosure widget that users can open and close
  • <summary> - Used as a summary, caption, or legend for the <details> element
  • <dialog> - Used to define a dialog box or other interactive component
  • <template> - Used to declare fragments of HTML that can be cloned and inserted in the document by script
  • <slot> - Placeholder inside a web component that users can fill with their own markup
  • <shadow> - Used to encapsulate a part of the DOM and keep it separate from the main document
